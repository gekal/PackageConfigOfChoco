# Chocoコンフィグ

## なぜこれを整理するなの

個人PC用のツールを整理する。

## パッケージのインストール

```powershell
# 対象フォルダに移動して、インストールする
# ⇒Long
choco install .\packages.config -y
# ⇒Short
cinst .\packages.config -y
```
